# Technical Writing

A repository for technical writing projects.

## Workflow

1. Create outline in `outlines/`
2. Refine outline with Claude
3. Draft article in `drafts/`
4. Review and iterate

## Directory Structure

- `outlines/` - Article outlines (rough and refined)
- `drafts/` - Draft articles in progress
- `medium-content/` - Exported Medium content (voice reference)
  - `posts/` - All posts (55 blog articles, 142 responses)
    - Blog posts are >5KB; responses are <5KB
    - Drafts are prefixed with `draft_`
  - Other directories contain metadata (bookmarks, claps, profile, etc.)
